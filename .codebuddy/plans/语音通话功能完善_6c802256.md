## 用户需求

点击 ChatPage 底部"📞"按钮，跳转到实时语音通话页面（CallScreen），进行双向语音通话，挂断后返回 ChatPage，并在消息列表中显示本次通话的时长记录。

## 产品概述

在现有 Omni 英语对话练习应用的基础上，完善语音通话功能。用户点击"语音通话"按钮后进入全屏通话页面，与 AI 助手进行实时双向语音对话，页面显示通话时长；挂断后自动返回聊天页面，并在消息列表中插入一条系统消息，显示本次通话时长。

## 核心功能

- 点击"语音通话"按钮，跳转到 `/call/:sessionId` 通话页面
- 通话页面实时显示通话时长（MM:SS 格式），每秒更新
- 通话页面显示实时对话字幕（用户说话内容和 AI 回复内容）
- 挂断电话后返回 ChatPage，并通过路由 state 传递通话时长
- ChatPage 接收通话时长，在消息列表末尾添加一条系统通话记录消息
- 通话记录消息以居中灰色胶囊样式展示，包含通话图标和格式化时长

## 技术栈

- 前端框架：React 19 + TypeScript + React Router DOM v7
- 样式方案：Tailwind CSS v4
- 实时通信：WebSocket（已有 `callWsUrl` 封装）
- 音频处理：Web Audio API + AudioWorklet（已有实现）

## 关键代码片段

### CallScreen 计时器实现

```
const [callDuration, setCallDuration] = useState(0);
const timerRef = useRef<ReturnType<typeof setInterval> | null>(null);

useEffect(() => {
  if (wsRef.current?.readyState === WebSocket.OPEN) {
    timerRef.current = setInterval(() => {
      setCallDuration(prev => prev + 1);
    }, 1000);
  }
  return () => {
    if (timerRef.current) clearInterval(timerRef.current);
  };
}, [wsRef.current?.readyState]);
```

### ChatPage 读取通话时长

```
const location = useLocation();

useEffect(() => {
  const state = location.state as { callDuration?: number } | null;
  if (state?.callDuration && state.callDuration > 0) {
    const dur = state.callDuration;
    const min = Math.floor(dur / 60);
    const sec = dur % 60;
    const content = `通话时长 ${String(min).padStart(2, "0")}:${String(sec).padStart(2, "0")}`;
    setMessages(prev => [...prev, {
      id: `call-${Date.now()}`,
      role: "system",
      content,
      source: "call",
    }]);
    navigate("/", { replace: true, state: {} });
  }
}, [location.state]);
```
